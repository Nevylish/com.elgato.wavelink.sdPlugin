## Installation

[Download the plugin](https://github.com/Nevylish/com.elgato.wavelink.sdPlugin/archive/refs/heads/main.zip), unzip and replace the folder at C:\Users\---\AppData\Roaming\Elgato\StreamDeck\Plugins
